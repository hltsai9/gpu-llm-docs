# 第二部分 —— GPU 架構

現在我們掀開引擎蓋。

第一部分告訴我們「為什麼」:GPU 是自助餐廳、不是牛排館,而 LLM 的數學剛好就是自助餐廳擅長的那種工作。第二部分要回答「怎麼做到的」:出餐員站在哪裡、倉庫長什麼樣子,以及為什麼這間廚房要這樣排布。

我們會用兩個互補的類比:

- **一座工廠,裡面有站點和倉庫。** 這是「空間」類比 —— 它讓你看到東西在晶片上「實體」住在哪裡。
- **一支廚房編制(kitchen brigade)。** 這是「組織」類比 —— 它讓你看到工作是怎麼被分派和協調的。

兩個都有用。當其中一個能把觀念講得更清楚時,我們就會切換到那一個。

## 讀完之後你會懂什麼

讀完第二部分之後,當你看到這樣一行文字:

> *H100 SXM5:132 個 SM、每個 SM 有 2048 條執行緒、80 GB 的 HBM3 記憶體頻寬 3.35 TB/s、50 MB 的 L2 快取、tensor core 可達 989 TFLOPs 的 BF16 算力。*

……你應該可以把它唸出來,而且每一個詞對你來說都有具體的意義。
